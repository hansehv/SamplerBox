Introduction:

This copy of the website gives overall information. The rest of this page is a quick start for tweaking the user interface.
  • The SamplerBox script sends several internal values as JavaScript variables in a generated script: SamplerBox_API. It can viewed at by simply displaying source in your webbrowser.
  • The SamplerBox-GUI.js javascript and SamplerBox.css style sheet combine these variables into building blocks / elements for use in the HTML pages.
  • Updatable variables, can be viewed by using the IDs named as ID_SB_<name> and viewed/updated in forms using IDs named input_SB_<name>.
  • The other variables and tables, as well as the necessary form statement etc, can be included by using ID's named elem_SB_<name>.
  • Just look at the sources the existing pages and examples below for the usage of these building blocks.
  • List of available building blocks
Note: Sofar it works as it should in Firefox, Chrome, Edge and Silk.
For Safari (I tested Windows and some older iPads), the slider implementation did not work well. You may get around this with Chrome or Firefox.
